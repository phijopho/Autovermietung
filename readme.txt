# Autovermietung

## Über das Projekt

Autovermietung ist ein Webprojekt, das im Rahmen einer Universitätsaufgabe entwickelt wurde. Es ermöglicht Benutzern das Ansehen und Buchen von Mietfahrzeugen über eine intuitive Benutzeroberfläche. Die Anwendung läuft lokal über XAMPP und verwendet eine SQL-Datenbank zur Verwaltung der Daten.

## Technologien

- PHP
- SQL
- HTML/CSS
- JavaScript (minimaler Einsatz)

## Setup

Um das Projekt einzurichten, folgen Sie diesen Schritten:

1. Installieren Sie XAMPP und starten Sie den Apache-Webserver sowie MySQL.
2. Importieren Sie die bereitgestellte SQL-Datei in Ihre MySQL-Datenbank.
3. Platzieren Sie das Projektverzeichnis im `htdocs`-Ordner von XAMPP und nennen Sie es "Autovermietung".
4. Öffnen Sie Ihren Browser und navigieren Sie zu `http://localhost/Autovermietung/index.php`.

## Empfohlene Browser

Die Anwendung wurde für die besten Ergebnisse in folgenden Browsern getestet:

- Google Chrome
- Opera
- Safari
- DuckDuckGo
